# TODO_APP
A simple full stack todo web app with testing using Typescript. 
This is used to demonstrate the Cypress end-to-end testing framework. 
---

## Commands

Start Frontend Server
> npm start

Start Backend Server
> npm start

Start Cypress
> run cy:open
